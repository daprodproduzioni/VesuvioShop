# README - Setup Bot Python

1. Installa dipendenze:
   cd bot
   uv pip install -r requirements.txt

2. Configura bot token in main.py (già inserito)

3. Run:
   uv run python main.py

4. BotFather setup:
   - /newapp → "VesuvioQuanticoShop"
   - Menu Button → "Launch App" → URL GitHub Pages
   - Payments → attiva Stars

5. Test:
   - /start nel bot
   - Apri Mini App
   - Testa checkout (test environment)
