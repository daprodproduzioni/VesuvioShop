# Vesuvio Quantico - Mini App

Telegram Mini App per l'album "Vesuvio Quantico" - 34 canzoni napoletane rivisitate in chiave AI.

## 🌋 Info

- **Album:** Vesuvio Quantico
- **Tracce:** 34 canzoni classiche napoletane
- **Prezzo:** 100 ⭐ (Telegram Stars)
- **Demo:** [/start su @VesuvioQuanticoBot](https://t.me/VesuvioQuanticoBot)

## 📁 Struttura

```
mini-app/
├── index.html   # Pagina principale
├── style.css    # Stili (tema volcano)
├── app.js       # Logica Telegram WebApp
└── README.md    # Questo file
```

## 🚀 Deploy su GitHub Pages

1. Vai su **Settings** → **Pages**
2. Source: `main` branch
3. Folder: `/ (root)`
4. Save

URL: `https://daprodproduzioni.github.io/VesuvioShop/`

## 🔗 Integrazione Bot

1. Apri **BotFather** (@BotFather)
2. `/mybots` → seleziona `VesuvioQuanticoBot`
3. **Bot Settings** → **Menu Button**
4. Configura URL: `https://daprodproduzioni.github.io/VesuvioShop/`
5. Test: `/start` → pulsante 🛒 "Apri Negozio"

## 💳 Pagamenti

I pagamenti funzionano tramite **Telegram Stars**:
- Album: 100 ⭐
- Founder Pass: 300 ⭐
- Abbonamento: 30 ⭐/mese

Il bot gestisce gli invoice via Bot API.

## 🛠️ Tech Stack

- HTML5, CSS3, Vanilla JS
- [Telegram WebApp SDK](https://core.telegram.org/bots/webapps)
- GitHub Pages (free hosting)
